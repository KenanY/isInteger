# isInteger

[![Dependency Status](https://gemnasium.com/KenanY/isInteger.png)](https://gemnasium.com/KenanY/isInteger)

isInteger [component](https://github.com/component).

## Installation

``` shell
$ component install KenanY/isInteger
```

## API

### isInteger(x)

Check is `x` is an integer.

``` javascript
isInteger(1254);
// => true
```

## Development

``` shell
$ npm install
$ make
```
